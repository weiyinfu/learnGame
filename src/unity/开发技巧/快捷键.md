# 设置自动刷新

在settings里面把autoRefresh关掉，使用Command+R快捷键。

避免重复刷新。

# Unity的三处设置

1. Unity编辑器全局设置：在Settings里面

2. Unity的BuildSettings：编译设置

3. Unity的ProjectSettings：项目设置，内容最为丰富，包括PlayerSettings
